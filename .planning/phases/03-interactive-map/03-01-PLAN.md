    - Accept `filtered_schools: Memo<Vec<School>>` as a prop
    - Create a `NodeRef::<leptos::html::Div>::new()` for the map container div
    - Store the Leaflet Map instance in a `StoredValue<Option<leaflet::Map>>`
    - Store current markers in a `StoredValue<Vec<leaflet::CircleMarker>>`

    **Map initialization Effect (runs once):**
    - Guard: `if map_instance.get_value().is_some() { return; }`
    - Get container from NodeRef: `let Some(container) = map_ref.get() else { return; }`
    - Cast to web_sys::HtmlElement: the Leptos HtmlElement<Div> derefs to web_sys::HtmlDivElement. Use `&container` or `container.deref()` -- try direct pass to `Map::new_with_element`. If that doesn't compile, use `wasm_bindgen::JsCast::unchecked_ref::<web_sys::HtmlElement>(&container)`.

      - max_zoom: 18
    - Store map in StoredValue

    **Marker update Effect (runs on each filtered_schools change):**
    - Get map from StoredValue; early return if None
    - Read `filtered_schools.get()`
    - Remove all old markers: iterate `markers.get_value()`, call `.remove()` on each
    - For each school with `coords.is_some()`:
      - Create `LatLng::new(coords.lat, coords.lng)`
      - Determine pin color using `profile_color_for_map()` (see below)
      - Create CircleMarker options via `js_sys::Object` + `js_sys::Reflect::set`:
        - radius: 8.0
        - fillColor: the profile color string
        - color: "#fff"
        - weight: 2.0
        - opacity: 1.0
        - fillOpacity: 0.85
      - Create `CircleMarker::new_with_options(&latlng, &options.into())`
      - Build popup HTML via `build_popup_html(&school)` (see below)
      - Call `marker.bind_popup(&JsValue::from_str(&popup_html))` -- use `bind_popup` (check leaflet crate 0.5 API: it may be `bind_popup` or `bind_popup_with_options`. If only `bind_popup_with_options` exists, pass `&JsValue::NULL` for options.)
      - Call `marker.add_to(&map)` (marker.add_to takes a &Map or a &JsValue)
      - Collect marker in new_markers Vec
    - Collect all LatLngs for bounds
    - Fit bounds (per D-17):
      - 0 markers: `map.set_view(&LatLng::new(52.52, 13.405), 11.0)` (reset to Berlin default)
      - 1 marker: `map.set_view(&latlngs[0], 14.0)`
      - 2+ markers: create `LatLngBounds` from first two points, extend with rest, call `map.fit_bounds(&bounds)`
    - Store new_markers in StoredValue
